# Font Deck
Font Deck is a macOS desktop utility for inspecting and previewing locally available fonts, and is designed for quick side-by-side comparison of font families using custom sample text.

## ⚠️ Disclaimer
- This app is a product of cognitive automation. Involved models include: GPT 5.4.
- We urge the avoidance of using this app. We are not responsible for any result. We do not promise anything.

## Requirements
- macOS on Apple silicon Mac
- Traditional Chinese literacy

## Stack
- Electron, React, Vite, TypeScript, fontkit
